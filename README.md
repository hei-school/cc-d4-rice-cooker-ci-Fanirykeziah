# Rice cooker management in Python.

## Style Guides 
Linter : 
```
pylint
```

To install pylint : 
```
pip install pylint
``` 

For unit test I used **unittest** 

## Installation
Installing dependencies 
```
pip install pylint
pip install unittest
```

## Usage
To running application : 
```
python RiceCooker.py
```

Running test : 
```
python -m unittest discover -s tests
````
