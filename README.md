# Rice cooker management in Typescript 
This project in Typescript don't have set warm mode , set steam mode.

## Standard 
Linter : 
```
ESLint
```

For unit test I used **Jest** 

## Installation
Installing dependencies 
```
npm install
```

## Usage
To running application : 
```
tsc main.ts
node main.js
```

Running test : 
```
npm test
````
