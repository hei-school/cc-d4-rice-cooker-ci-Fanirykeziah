# Rice cooker management In Ruby 

## Style Guides
Linter : 
```
Rubycop 
```

For unit test I used **RSpec** 

## Installation
Installing dependencies 
```
bundle install
```

## Usage
To running application : 
```
ruby rice_cooker.rb
```

Running test : 
```
rspec
````
